Installation of ACME

To add ACME to your project just link the file acme.cmake (in acme/cmake/) to your topmost CMakeList.txt.
Therefor use the cmake command INCLUDE() in the CMakeList.txt of your project source directory.

Installation of Doxygen

In order to create a build report, ACME requires the additional software Doxygen.
Just install Doxygen into the standard directory of your operating system, ACME will autonomously search for executables.
If ACME couldn`t find Doxygen, you can add the path to the executable into the cache variable "DOXYGEN_EXECUTABLE" in the cmake GUI.

Doxygen Binaries

Installation of Graphviz

ACME additionally requires the software "dot" to create the dependency graph of the build report.
Install Dot into the standard directory of your operating system. Therefor you can use the binaries of the programm "Graphiz" (Graphviz contains Dot).
If ACME couldn`t find Dot, you can add the path to the executable into the cache variable "DOXYGEN_DOT_EXECUTABLE" in the cmake GUI.
